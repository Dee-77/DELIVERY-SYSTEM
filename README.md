# DELIVERY SYSTEM (JAVA OOP PROJECT)
This is a simple Delivery Management System built using Java Object-Oriented Programming concepts.
It demonstrates inheritance, polymorphism, and abstraction using different delivery types such as Food Delivery, Grocery Delivery, and Pharmacy Delivery.

Each delivery type has its own delivery behavior and fee calculation based on distance.

CONCEPTS USED:
- Inheritance
- Polymorphism (Method Overriding)
- Abstraction (Abstract Class)
- Dynamic Method Dispatch
- Arrays of Parent Class Type

PROJECT STRUCTURE:
Main.java
Delivery.java
FoodDelivery.java
GroceryDelivery.java
PharmacyDelivery.java

FEATURES:
- Different delivery types (Food, Grocery, Pharmacy)
- Custom delivery behavior for each type
- Fee calculation based on distance
- Clean OOP design using polymorphism
